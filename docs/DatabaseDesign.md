# SmartGuard Database Design

## Devices Table

Stores all registered SmartGuard devices.

| Field | Type |
|---------|---------|
| id | Integer |
| device_id | String |
| name | String |
| location | String |
| status | String |
| created_at | DateTime |

Example:

SG001
Fiber Cabinet 1
Windhoek CBD
online

---

## Events Table

Stores all activity detected by devices.

| Field | Type |
|---------|---------|
| id | Integer |
| device_id | String |
| event_type | String |
| timestamp | DateTime |

Example:

door_opened

motion_detected

camera_capture

---

## Alerts Table

Stores alerts generated by SmartGuard.

| Field | Type |
|---------|---------|
| id | Integer |
| device_id | String |
| alert_level | String |
| message | String |
| timestamp | DateTime |

Example:

HIGH

Telecom cabinet opened after hours
